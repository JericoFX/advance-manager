# Brainstorming de mejoras futuras

- **Modo de inspección avanzada**: mostrar histogramas y análisis de canales para facilitar ajustes cromáticos antes de exportar.
- **Comparador en paralelo**: dividir la vista previa en dos paneles para comparar original y reemplazo, con sincronización de zoom.
- **Exportación a paquetes modulares**: generar automáticamente una estructura de carpetas lista para distribuir en mods, incluyendo documentación y logs.
- **Historial de reemplazos**: mantener una lista cronológica de cambios con posibilidad de revertir a versiones anteriores sin reimportar archivos.
- **Integración con editores externos**: abrir la textura en un editor configurado y detectar guardados para refrescarla en la interfaz.
- **Validaciones específicas por juego**: añadir perfiles que definan formatos aceptados por cada título y alerten sobre incompatibilidades particulares.
- **Temas personalizables**: permitir elegir entre distintos temas claros/oscuro y exponer ajustes de fuente y tamaño de iconos.
- **Modo scriptable**: exponer un API REST local o una interfaz CLI separada para automatizar exportaciones sin perder la nueva experiencia gráfica.
